# Fadex25
A Node.js and React application for Perps market data, backtesting, and live trading using Orderly Network APIs and SQLite.

## Structure
- `server.ts`: Express server for hosting React UI and API endpoints.
- `api/`: Scripts for Orderly API calls.
- `db/`: SQLite database configuration.
- `indicators/`: Indicator calculation scripts.
- `fadex25-ui/`: React UI with MarketData, LiveTrade, and Backtester pages.
